# @ch3sh1r's dotfiles

## Installation

*WARNING*! Next steps will remove current configuration files.

### Debian based

```bash
sudo apt update
sudo apt install vim git fish tmux make
git clone --recursive https://github.com/ch3sh1r/dotfiles ~/.dotfiles
cd ~/.dotfiles
make
```

### macOS

```bash
brew install vim git fish tmux make
git clone --recursive https://github.com/ch3sh1r/dotfiles ~/.dotfiles
cd ~/.dotfiles
make
```
